# 용어 사전

| 한국어            | 영어              | 설명                                                                       | 축약어  | 
|----------------|-----------------|--------------------------------------------------------------------------|------|
| 회원             | Member          | 서비스를 이용하는 사용자로 강의를 수강할 수 있다                                              |      |
| 강사             | Instructor      | 강의를 만들고 제공하는 회원                                                          |      |
| 수강생            | Student         | 강의를 듣는 회원                                                                |      |
| 강의             | Course          | 강사가 등록하는 강의                                                              |      |
| 강의 등록          | Open            | 강사가 강의를 등록할때 표현하는 단어                                                     |      |
| 수강             | Enrollment      | 회원이 강의를 학습하는 것. 수강 신청을 통해서 수강할 수 있다                                      |      |
| 수강 신청자         | applicant       | 수강 신청 시 회원을 가리키는 단어 예) applicantId = 신청자 ID                              |      |
| 수강 신청          | Apply           | 수강 신청을 가리키는 단어                                                           |      |
| 패스워드           | Password        | 비밀번호                                                                     |      |
| 인코딩 된 패스워드     | EncodedPassword | 비밀번호를 인코딩 한 값. 사람이 알아볼 수 없는 문자열로 보여진다                                    |      |
| 인코딩 되지 않은 패스워드 | RawPassword     | 인코딩 하지 않은 패스워드를 표현할 때 사용하는 단어                                            |      |
| 새로운 값          | New[값]          | 이 애플리케이션에서 앞으로 '이전 값을 새 값으로 교체한다'라는 표현은 모두 newXxx 로 표현한다. 예) newPassword |      |
| 이메일            | Email           | 이메일                                                                      |      |
| 휴대폰 번호         | PhoneNumber     | 휴대폰 번호                                                                   |      |
| 회원 유형          | MemberRole      | 회원 유형을 나타낸다. 예) INSTRUCTOR, STUDENT                                      | role |                                     
| 회원 가입          | Register        | 회원 가입을 표현하는 단어                                                           |      |
| 변경             | Change          | 값을 변경할때 붙이는 Prefix. 예) changePassword, changeRole, ...                   |      |
| 명세             | Specification   | 명세서를 표현하는 단어 예) CourseModifyControllerSpecification                      | Spec |
